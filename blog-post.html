Featured image

This is the lead paragraph of the article body. When imported into WordPress, everything inside .post-body becomes the post content, so use standard semantic HTML and the WP importer will convert it to Gutenberg blocks cleanly.

A Section Heading

Use h2 for major sections and h3 for sub-sections. Keep paragraphs short and scannable. WordPress will convert each block-level element to its matching Gutenberg block: paragraphs to Paragraph blocks, headings to Heading blocks, lists to List blocks, blockquotes to Quote blocks, and so on.

  • Bulleted lists translate to the List block.
  • Each item becomes a child list-item.
  • Nesting works too.

Subsection Heading

Inline elements like bold, italic, and links import as expected. Code samples in code tags stay inline; longer blocks should use pre > code.

A pull quote, often used to highlight a key statistic or sentence. Renders as a Quote block in WordPress.

Wrapping Up

Close out the article with a takeaway or call-to-action paragraph. The CTA strip below the article is part of the theme, not the post content, it appears on every blog post.
